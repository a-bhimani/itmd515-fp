About About Student Event Portal

The Student Event Portal is developed by Ankit Bhimani to fullfil the final project submission for ITMD 515 - Advanced Software Programming course based on Java EE.

Ever missed an important event at the university just because you were unaware about it or could not find it on the university website? The project Student Event Portal is designed to bridge this gap between the university departments, events they are holding and the students. It can be a portal for students to keep themselves updated about the event, its timings and venue, and interact with the event organizers and other students.
